Objetivos

  • Aprender a extraer información de los datos, especialmente datos masivos (“big data”)

  • Enfoque práctico: aprenderemos técnicas y cómo aplicarlas para posteriormente usar la información en crear valor

  • Tres Componentes:

    1. Nociones de Programación

    2. Obtención y Procesamiento de Datos

    3. Modelos de Aprendizaje Estadístico

Información para las clases

  • Es muy recomendable que cada estudiante trabaje en clase con su propio ordenador de forma habitual

    • Nota: considerad el uso de regletas si necesitáis enchufar vuestros dispositivos (especialmente, si sus baterías tienen poca duración).

  • ANTES DE comenzar las clases, vuestro ordenador deberá tener instalados los siguientes programas de software (totalmente gratuitos): el lenguaje de programación R y el interfaz-editor para trabajar con R llamado RStudio.

    • OJO: es conveniente tener el sistema de vuestros ordenadores actualizados antes de instalar nada.

  • Podéis descargar R desde https://cran.r-project.org/ siguiendo las instrucciones para vuestro sistema operativo.

    • para Windows, entrad en “Download R for Windows”, luego en “base” y allí encontraréis la versión actual “Download R-4.3.1 for Windows”.
    • para MacOS, entrad en “Download R for (Mac) OS X”, y allí la versión correspondiente a vuestra versión (para las últimas versiones de MacOS “R-4.3.1-x86_64.pkg” o “R-4.3.1-arm64.pkg”, según vuestro ordenador). NOTAD que puede ser necesario instalar XQuartz.
    • para Linux, seguid las instrucciones (típicamente a través de vuestro sistema de instalación).

  • Podéis descargar RStudio desde https://posit.co/download/rstudio-desktop/

    • Desplazaros hacia abajo en la página hasta encontrar una tabla con enlaces para descargas
    • Elegid el enlace de descarga para vuestro sistema.
      • En Windows, usad el instalador ejecutable .EXE (no .ZIP) y durante la instalación elegid la opción de 64 bits.

  • NOTA: existe la posibilidad de usar R y RStudio de forma online, sin instalar nada en el ordenador (o incluso usando una tablet). Se puede encontrar en https://posit.cloud/ . Sin embargo, no la recomendamos para su uso habitual; en particular, algunas cosas son ligeramente diferentes, es más lenta, depende de conexión a internet y, aunque es gratuita, requiere registrarse.

  • Más adelante (dependiendo del tiempo e intereses de los alumnos) se considerará software adicional
